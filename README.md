# Sample QA Website

This repository contains a small static storefront website intended for testing
another feedback or QA tool.

## Run locally

```bash
npm start
```

Then open:

```text
http://localhost:4173
```

## Notes

- The site is intentionally simple and dependency-free.
- A few non-breaking content and layout issues are intentionally present so a
  QA or review tool has something to detect.
